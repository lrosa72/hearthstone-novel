| 项目 | 内容 |
|------|------|
| 章节标题 | 镜 |
| 预估字数 | 3000 ± 400 字 |
| 本章定位 | 兔子洞深处——"镜像之墙"揭示双重自我的奇观；谢知白（玉版白）现实身份交汇 |

## 场景清单

### 场景 1/3：集市深处的墙（地下世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 地下集市最深处，午后 |
| 出场人物 | 余烬（独自）、背景人群 |
| 场景目标 | 奇观：白描镜像之墙——卡牌世界与现实世界之间的夹缝 |
| 核心事件 | 余烬独自逛到集市深处，发现一面巨大的石墙——不是镜子，但能映出人的倒影。赵一行说这是"夹缝之墙"：站在这里能看到自己在炉石世界的形象。余烬看到了——镜中的自己不是余烬，是鱼刺。黑框眼镜还在，但眼神不一样：更犀利，更懒散，更像那个在酒馆里偷人牌的脏牧。鱼刺在镜中朝他笑了一下，像是等了很久。 |
| 情感基调 | 震撼、毛骨悚然的亲切、自我认知的摇晃 |
| 信息增量 | 魂穿的真正意义——不是"进入"炉石世界，而是炉石世界一直有一个他，万胜只是把两个他连起来了 |
| 预计字数 | ~1000 字 |

### 场景 2/3：镜前的另一个人（地下世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 镜像之墙前，午后 |
| 出场人物 | 余烬（鱼刺）、谢知白·玉版白 |
| 场景目标 | 谢知白现实身份与炉石身份的首次现实交汇——她也在镜前 |
| 核心事件 | 余烬正盯着镜中的鱼刺出神，旁边传来一个熟悉的声音——"你也看到自己了？"谢知白站在几步外，素颜，白玉菩提在腕，镜中映出的是玉版白：冰霜绕身，法杖在手。两人在现实里第一次见面，但已经在酒馆里互怼过一轮。谢知白说她在专栏里写过一篇关于"夹缝之墙"的文章——她比余烬更早发现这个地方。两人并肩看镜，沉默中有种奇怪的默契。 |
| 情感基调 | 奇妙的同步感、宿敌变"同类"的微妙 |
| 信息增量 | 谢知白发现这个世界比余烬早——她的专栏写作是掩护，真实目的是研究卡牌世界；F-003推进：玉版白的双重身份不再只是游戏里的 |
| 预计字数 | ~1000 字 |

### 场景 3/3：镜中的倒影动了（地下世界→现实过渡）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 镜像之墙前→卡牌店，傍晚 |
| 出场人物 | 余烬、谢知白、赵一行（店中等候） |
| 场景目标 | 钩子——镜中的鱼刺做了一个余烬没做的动作 |
| 核心事件 | 余烬准备离开时，最后看了一眼镜子。镜中的鱼刺没有转身——他站在原地，举起手里的卡牌，朝他亮了一张他没见过的卡。卡面模糊，但漩涡背面是真的。余烬回到卡牌店，赵一行在喝茶。他说"看到什么了？"余烬说"看到我自己。还有一张我没见过的卡。"赵一行放下茶杯，表情第一次变得认真。 |
| 情感基调 | 悬疑、不安的期待 |
| 信息增量 | 镜中的鱼刺似乎拥有余烬还没有的卡——暗示炉石世界的"他"在独自成长？还是镜像之墙在预示未来？ |
| 预计字数 | ~900 字 |

## 钩子设计

**开篇钩子**：赵一行说集市深处有一面墙，站上去能看到自己。余烬以为是面镜子。赵一行说："镜子照的是脸，那面墙照的是别的东西。"

**结尾钩子**：余烬把镜中那张卡的描述告诉赵一行——卡面模糊，但漩涡背面是真的。赵一行放下茶杯，杯底和木柜台碰出沉闷的一声。"那是……"他没说完，起身从柜台底下翻出一本旧账本，翻开其中一页。上面画着一张卡，和余烬描述的一模一样。
