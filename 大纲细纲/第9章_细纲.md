## 场景清单

### 场景 1/3：酒馆里的赵一行（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 炉石酒馆，夜晚 |
| 出场人物 | 鱼刺、赵一行（炉石世界形象）、酒馆老板 |
| 场景目标 | 赵一行在炉石世界的身份揭示——他不是"访客"，是酒馆的熟客 |
| 核心事件 | 鱼刺照常进入酒馆，发现赵一行坐在角落——不是以导师身份，是以一个老玩家的身份。他在酒馆里的形象跟现实不同：皮手套还在，但缺小指的手上多了一枚戒指，戒指上嵌着一颗暗金色的晶石。酒馆老板给他倒的不是茶，是麦酒。老板叫他"老赵"——跟现实里一样。鱼刺意识到赵一行在炉石世界也有十年以上的资历。 |
| 情感基调 | 好奇、轻微的震撼、"原来你早就在这里了" |
| 信息增量 | 赵一行在炉石世界是资深玩家——他隐退后把时间花在了这里；双世界时间差让他在这边积累了远超现实的经验 |
| 预计字数 | ~1000 字 |

### 场景 2/3：新手任务（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 酒馆→冒险模式入口，夜晚 |
| 出场人物 | 鱼刺、赵一行 |
| 场景目标 | 第一个新手任务——赵一行教余烬"精神力控制"的基础 |
| 核心事件 | 赵一行说"你昨天在桥上差点掉下去——是因为精神力散了。今天教你基本功。"他带鱼刺到酒馆后门——冒险模式的入口。新手任务不是对战，是一个简单的精神力控制练习：用一张白卡凝出一颗光球，让它浮在掌心不散。赵一行说"你在现实里具现卡牌，精神力是往外轰的。但在桥上、在炉石世界里，你得学会'收'——像呼吸，有出有进。"余烬试了三次。第一次光球炸了。第二次光球浮了三秒散了。第三次——他想起石壁里鱼刺朝他笑的那一下，想起桥上脚下流过的城市灯火，光球稳稳浮在掌心。 |
| 情感基调 | 训练感、渐进的成就感、安静 |
| 信息增量 | 精神力控制的"收放"原理；冒险模式的入口在酒馆后门 |
| 预计字数 | ~1100 字 |

### 场景 3/3：戒指和账本（炉石世界→现实过渡）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 酒馆→现实（余烬醒来），清晨 |
| 出场人物 | 鱼刺/余烬、赵一行、酒馆老板 |
| 场景目标 | 赵一行那枚戒指的来历暗示；余烬回到现实发现新变化 |
| 核心事件 | 训练结束后，鱼刺问赵一行那枚戒指是什么。赵一行说"很久以前的东西。跟那张'门·桥'的卡是同一套。"他没展开说。鱼刺醒来，窗外天刚亮。他抬手——掌心没有光球，但指尖有一种新的感觉：不是精神力耗尽的虚脱，是一种"收住了"的踏实。他试着凝奥术飞弹——三颗飞弹凝出来，比之前更稳，蓝光不再忽闪。 |
| 情感基调 | 安静的满足、对戒指的好奇、新能力的踏实感 |
| 信息增量 | 赵一行戒指与"门·桥"卡同源——F-009；余烬精神力控制初步入门 |
| 预计字数 | ~800 字 |

## 钩子设计

**开篇钩子**：鱼刺推开酒馆的门，看见赵一行坐在角落。不是现实里那个泡陈皮茶的精瘦老头——炉石世界里的他，皮手套还在，但缺小指的手上多了一枚暗金色的戒指。

**结尾钩子**：余烬站在窗前，看着掌心凝出的三颗飞弹——蓝光稳得像三颗嵌在空气里的宝石。手机震了。苏新晖：「今天有空吗？有件事想跟你说。」他盯着那行字看了五秒。苏新晖从不主动找他。
