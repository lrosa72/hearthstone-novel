| 项目 | 内容 |
|------|------|

| 预估字数 | 3000 ± 400 字 |

## 场景清单

### 场景 1/3：卡牌店后门（现实→过渡）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 恒记卡牌店，午后 |
| 出场人物 | 余烬、赵一行 |
| 场景目标 | 赵一行揭示"门"的存在；过渡到地下世界 |
| 核心事件 | 赵一行说"今天教你第一课——怎么进门"。掀开卡牌店后墙的旧海报（2014年炉石经典版），后面是一扇嵌在砖墙里的暗门。门上没有锁，只有一个凹槽——卡牌大小的凹槽。余烬把自己的奥术飞弹卡插进去，门开了。 |
| 情感基调 | 神秘、仪式感、像第一次穿过九又四分之三站台 |
| 信息增量 | 卡牌店的真正功能不是卖卡——是通往地下世界的入口之一 |
| 预计字数 | ~900 字 |

### 场景 2/3：地下集市（卡牌地下世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 地下卡牌集市，午后 |
| 出场人物 | 余烬、赵一行、各路能力者（背景群像）、顾未明（首次出场） |
| 场景目标 | 奇观轰炸——白描地下集市的烟火气与异世界感 |
| 核心事件 | 赵一行带余烬逛集市：烛光与晶石照明、交易用的"尘"（奥术之尘）、各种奇观摊位（有人在用卡牌烤红薯、有人在展示会唱歌的随从、有人在兜售"能让你多睡一小时"的时间卡——赵一行说是骗子）。顾未明穿便服出现——她不是来抓人的，是在做例行巡查。她注意到余烬，在笔记本上写了点什么，但没说话。赵一行低声说"清理者，别惹她"。 |
| 情感基调 | 目不暇接、好奇、轻微的紧张（顾未明的存在像一根隐形的线） |
| 信息增量 | 地下世界的货币（尘）、日常生态、清理者的存在方式（不是警察，更像审计员） |
| 预计字数 | ~1300 字 |

### 场景 3/3：笔记本上的编号（地下集市→现实过渡）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 地下集市出口，傍晚 |
| 出场人物 | 余烬、赵一行、顾未明（背影） |
| 场景目标 | 顾未明留下悬念；余烬第一次意识到自己被"记录在案" |
| 核心事件 | 余烬离开集市时，在出口的石壁上看到一行粉笔字——「编号001，首次进入，待观察。」笔迹工整，像公文。赵一行看了一眼，说"她给你编号了。好事，编号说明你在规则里——规则外的人，她不写字，直接动手。"余烬回头，顾未明的背影正消失在集市的烛光深处。 |
| 情感基调 | 轻微的寒意、被注视的不安、但混合着兴奋 |
| 信息增量 | 顾未明给余烬编号001——呼应人物设定中"异常编号001"；赵一行对清理者的评价"规则之内就安全" |
| 预计字数 | ~700 字 |

## 钩子设计

**开篇钩子**：赵一行把墙上那张贴了十年的炉石海报揭下来。海报后面不是墙——是一扇门。

**结尾钩子**：余烬走出集市时回头看了一眼。石壁上的粉笔字还在——「编号001，首次进入，待观察。」他伸手想擦，赵一行按住他手腕："别擦。擦了就是心虚，心虚才出规则。"
