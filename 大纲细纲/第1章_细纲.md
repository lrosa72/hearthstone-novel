# 第1章 细纲 — "时空旅人"

## 基本信息

| 项目 | 内容 |
|------|------|
| 章节标题 | 时空旅人 |
| 预估字数 | 2000 ± 200 字 |

## 场景清单

### 场景 1/3：废柴的日常（现实）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 城中村合租房，傍晚 |
| 出场人物 | 余烬、刘胖子（电话）、苏新晖（电话催租） |
| 场景目标 | 建立主角形象，展示困境，制造共情 |
| 核心事件 | 余烬被房东电话催租，翻遍抽屉找钱，意外翻到神秘卡牌 |
| 情感基调 | 压抑中带着毒舌吐槽（以幽默化解绝望） |
| 信息增量 | 主角的处境、性格、毒舌风格全部建立 |
| 预计字数 | ~600 字 |

### 场景 2/3：神秘卡牌（现实→过渡）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 同上，稍晚 |
| 出场人物 | 余烬（独白） |
| 场景目标 | 建立神秘感，制造悬念，触发魂穿 |
| 核心事件 | 余烬研究卡牌"时空旅人"，被卡牌烫伤手指，意识模糊 |
| 情感基调 | 好奇→困惑→恐惧 |
| 信息增量 | 卡牌的基本信息，暗示这不是普通的游戏周边 |
| 预计字数 | ~600 字 |

### 场景 3/3：首次魂穿（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 炉石传说的酒馆（初始场景） |
| 出场人物 | 余烬、酒馆老板（NPC） |
| 场景目标 | 展示炉石世界的真实感，完成本章钩子 |
| 核心事件 | 余烬发现自己身处一个真实的炉石酒馆，酒馆老板递给他一副新手卡组 |
| 情感基调 | 震撼→兴奋→"这什么离谱展开" |
| 信息增量 | 炉石世界是真实的、有实体、有NPC |
| 预计字数 | ~800 字 |

## 钩子设计

**开篇钩子**（前 300 字）：
> 以余烬接到房东催租电话开场，用毒舌吐槽直接抓住读者——"我的人生就像炉石的标准模式：每年都在退环境，而我是那张被淘汰的基础卡。"

**结尾钩子**（最后一段）：
> 酒馆老板把新手卡组推到他面前："欢迎来到炉石酒馆，新人。你的第一场对战——现在开始。"余烬低头看着自己手中的卡牌——它们在他掌心里微微发热，就像活着的一样。
