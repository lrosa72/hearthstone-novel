## 场景清单

### 场景 1/3：不将就的自由人（现实）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 二线城市公寓，傍晚 |
| 出场人物 | 余烬、刘胖子（电话）、苏新晖（微信催租，一笔带过） |
| 场景目标 | 建立"数字游民 + 不将就"人设，用社会压力强化核心冲突，立住毒舌声线 |
| 核心事件 | 余烬在家一边接外包一边打炉石；母亲来电催他"找个稳定工作将就将就"；前女友那句"你这么聪明，为什么不肯将就一下"在脑中回响；刘胖子电话扯淡；苏新晖微信"房租该交了"一笔带过 |
| 情感基调 | 表面松弛，底色是被"将就"声音包围的不甘——用毒舌吐槽把压力化解成笑点 |
| 信息增量 | 余烬现状（自由职业、时间自由）、核心冲突（不将就）、毒舌风格、牧师+盗贼的游戏身份 |
| 预计字数 | ~800 字 |

### 场景 2/3：牧师万胜（现实→过渡）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 同上，深夜 |
| 出场人物 | 余烬（独白 + 操作） |
| 场景目标 | 展示炉石痴迷与真实实力，铺垫"资格认证"触发魂穿 |
| 核心事件 | 余烬冲牧师天梯，离"万胜"成就只差最后几胜。关键局对手天胡，他用脏牧的思维窃取 + 精神控制极限翻盘，屏幕弹出"万胜"字样——下一秒，炉石 logo 开始发光 |
| 情感基调 | 专注 → 紧张 → 狂喜 → 诡异（logo 发光，他以为是屏幕坏了） |
| 信息增量 | "牧师万胜"是魂穿的资格认证；logo 发光是触发信号（呼应世界观设定） |
| 预计字数 | ~700 字 |

### 场景 3/3：首次魂穿（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 炉石传说的酒馆（初始场景） |
| 出场人物 | 余烬、酒馆老板（NPC） |
| 场景目标 | 展示炉石世界的真实感，完成本章钩子 |
| 核心事件 | 余烬的意识被拉入一个真实的炉石酒馆，木桌、火炉、喝酒的"玩家"全是实体。酒馆老板认出他牧师万胜的身份，递来一副卡组："万胜的牧师，欢迎回来。"余烬低头——卡牌在他掌心微微发热，像活的一样 |
| 情感基调 | 震撼 → 兴奋 → "这什么离谱展开" |
| 信息增量 | 炉石世界是真实的、有实体、有 NPC；卡牌可具现的雏形；老板似乎"认识"他的战绩 |
| 预计字数 | ~700 字 |

## 钩子设计

**开篇钩子**（前 300 字）：
> 以余烬接到母亲催"将就"的电话开场，用毒舌自嘲直接抓读者——"我的人生就像炉石的标准模式：每年都在退环境，而我是那张被淘汰的基础卡。但基础卡怎么了？基础卡也有不让步的尊严。"

**结尾钩子**（最后一段）：
> 酒馆老板把卡组推到他面前："万胜的牧师，欢迎回来。你的第一场对战——现在开始。"余烬低头看着自己手中的卡牌，它们在他掌心里微微发热，就像活着的一样。
