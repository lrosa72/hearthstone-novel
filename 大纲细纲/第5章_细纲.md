### 场景 1/3：卡牌店的下午（现实）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 街角卡牌店，下午 |
| 出场人物 | 余烬、老赵 |
| 场景目标 | 老赵正式收徒（考选择）；展开卡牌世界结构 |
| 核心事件 | 余烬应第3章邀约上门。老赵不教操作，扔一道无解题残局，看他死磕还是找第三条路。余烬用炉石术语解构，给出老赵没想到的解法→老赵轻描淡写收为弟子 |
| 情感基调 | 从容、棋逢对手的默契 |
| 信息增量 | 老赵讲卡牌世界结构：天梯玩家/收藏家协会/卡牌猎人/清理者；双世界时差（现实1h=炉石3h） |
| 预计字数 | ~700 字 |

## 钩子设计

**开篇钩子**：老赵把一副残局推过来："解不开。说说你怎么选。"余烬盯着那堆卡，忽然笑了——这题他熟，炉石里最难的从来不是赢，是认输之前先找第三条路。

**结尾钩子**：高看倚着旧物店门框，指尖转着一串白玉菩提："小余烬，这世界大着呢。南边、北边、海边，都有酒馆。你一个能到处活着的人，迟早要上路的。"余烬回头，黄昏把她的影子拉得很长。
