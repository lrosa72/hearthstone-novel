# 第2章 细纲 — "新手试炼"

---

## 基本信息

| 项目 | 内容 |
|------|------|
| 章节标题 | 新手试炼 |
| 预估字数 | 2000 ± 200 字 |
| 本章定位 | 展开世界——揭示炉石世界的规则 |
| 核心冲突 | 陈墨 vs 新手对战对手（NPC） |

## 场景清单

### 场景 1/3：第一场对战（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 炉石酒馆对战桌 |
| 出场人物 | 陈墨、酒馆老板（裁判）、NPC对手 |
| 场景目标 | 展示卡牌对战的真实体验（不是点鼠标，是真的在战场上） |
| 核心事件 | 陈墨的第一场炉石对战——实体化的卡牌对战，随从真的站在战场上 |
| 情感基调 | 紧张→慌乱→找到感觉→逆袭获胜 |
| 信息增量 | 对战规则：卡牌具现为实体、英雄亲自指挥、胜负的代价 |
| 预计字数 | ~900 字 |

### 场景 2/3：获胜与奖励（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 酒馆内 |
| 出场人物 | 陈墨、酒馆老板 |
| 场景目标 | 获得第一张战利品卡牌，了解基本规则 |
| 核心事件 | 酒馆老板解释新手规则，陈墨获得第一张战利品卡牌"奥术飞弹" |
| 情感基调 | 兴奋、好奇、跃跃欲试 |
| 信息增量 | 卡牌获取机制、每日任务、职业选择 |
| 预计字数 | ~500 字 |

### 场景 3/3：醒来（现实）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 合租房，清晨 |
| 出场人物 | 陈墨 |
| 场景目标 | 制造悬念——炉石世界的经历是梦还是真的？ |
| 核心事件 | 陈墨在床上醒来，以为一切都是梦——直到他发现手里攥着那张"奥术飞弹"卡牌 |
| 情感基调 | 恍惚→震惊→狂喜 |
| 信息增量 | 卡牌可以从炉石世界带回现实——能力具现的伏笔 |
| 预计字数 | ~600 字 |

## 钩子设计

**开篇钩子**：直接从对战开始——"你的回合。"酒馆老板的声音把陈墨从懵逼中拉回来。

**结尾钩子**：陈墨攥着"奥术飞弹"卡牌的手在发抖。不是梦。卡片上的法术图案在晨光下微微闪烁，就像在对他说——来试试看。
