## 场景清单

### 场景 1/3：第一场对战（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 炉石酒馆对战桌 |
| 出场人物 | 鱼刺、酒馆老板（裁判）、法师对手（老油条玩家） |
| 场景目标 | 展示实体卡牌对战的真实体验 + 脏牧逆风翻盘 |
| 核心事件 | 鱼刺第一场实体对战，对手是个控场法师，把他按在地上摩擦。血线见底时，鱼刺用思维窃取偷到对方斩杀组件、精神控制抢回场面、暗言术灭清核心，一回合翻盘 |
| 情感基调 | 紧张 → 崩溃边缘 → 灵光一现 → 逆袭 |
| 信息增量 | 对战规则：随从实体化、英雄亲临指挥、胜者复制对手一张卡 |
| 预计字数 | ~900 字 |

### 场景 2/3：踢馆的来了（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 酒馆对战桌 |
| 出场人物 | 鱼刺、刀疤（战士，前期对手）、法师对手（旁观） |
| 场景目标 | 引入刀疤、制造第二场起伏、立住"对手非恶人" |
| 核心事件 | 刀疤看完第一场，嗤笑"赢了个小法师就飘了？"当场踢馆。战士狂暴铺场碾压鱼刺，鱼刺靠刚赢来的"奥术飞弹"炸开缺口 + 脏牧偷牌续命，险胜。刀疤被恶心到又服气："你这打法，脏得我想吐——但有效。" |
| 情感基调 | 得意 → 被碾压的慌 → 灵机 → 惨胜的爽 |
| 信息增量 | 刀疤=卡牌猎人/前期对手；战利品卡"奥术飞弹"正式到手；能力者之间会互相挑战 |
| 预计字数 | ~700 字 |

### 场景 3/3：获胜与醒来（炉石→现实）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 酒馆 → 合租房清晨 |
| 出场人物 | 余烬、酒馆老板、刀疤（离场多瞥一眼） |
| 场景目标 | 规则讲解 + 悬念收尾 |
| 核心事件 | 老板讲解：胜者复制对手卡（每日限一张）、每日任务、职业选择。余烬在床上醒来，以为梦——手里却攥着"奥术飞弹" |
| 情感基调 | 兴奋 → 恍惚 → 震惊 → 狂喜 |
| 信息增量 | 卡牌可从炉石世界带回现实（具现伏笔）；刀疤埋线 |
| 预计字数 | ~600 字 |

## 钩子设计

**开篇钩子**："你的回合。"酒馆老板的声音把鱼刺从懵逼里拽回来。对面坐着的玩家已经铺满一地传说随从。

**结尾钩子**：余烬攥着"奥术飞弹"卡牌的手在发抖。不是梦。卡片上的法术图案在晨光下微微闪烁，像在说——来试试看。
