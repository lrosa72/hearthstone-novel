# 第8章 细纲 — "桥"

---

## 基本信息

| 项目 | 内容 |
|------|------|
| 章节标题 | 桥 |
| 预估字数 | 3000 ± 400 字 |
| 本章定位 | 兔子洞最高处——"桥"的奇观；小高潮不是战斗，是看见整个世界版图 |
| 核心冲突 | 无战斗。张力来自"世界比想象中大得多"的震撼与"选择"的暗示 |
| 本章转折 | 从"地下世界是一个集市"到"每个城市都有入口——这个世界是一座群岛" |

## 场景清单

### 场景 1/3：高看的账本（旧物店→地下世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 高看旧物店→地下集市，午后 |
| 出场人物 | 余烬、高看、赵一行 |
| 场景目标 | 高看揭示"桥"的存在——连接各地卡牌节点的隐秘通道 |
| 核心事件 | 高看翻开她的账本——不是普通的账本，每一页都是一个城市的卡牌节点坐标。北京的地下酒馆藏在潘家园旧货市场底下，成都在宽窄巷子的老茶馆后面，杭州在西湖边的碑林里。她说这些节点之间有"桥"相连——不是物理的桥，是精神力凝成的通道。余烬想起高看第5章说的"南边、北边、海边都有酒馆"——现在她给了他地图。 |
| 情感基调 | 开阔、向往、地图在眼前展开的感觉 |
| 信息增量 | 卡牌世界的跨地域网络；"桥"的存在让"在路上"从比喻变成可能 |
| 预计字数 | ~1000 字 |

### 场景 2/3：走在桥上（"桥"的奇观）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 桥（精神力通道），午后 |
| 出场人物 | 余烬、高看（引路）、谢知白（同行） |
| 场景目标 | 奇观高潮——站在桥上看见整个世界 |
| 核心事件 | 高看带余烬走上桥。桥是透明的——脚下是无数城市的缩影在流动：北京的鼓楼、成都的火锅店、杭州的西湖、上海的弄堂……每个城市都有一个小小的光点，那是卡牌节点的位置。谢知白也来了——她带了笔记本，在记坐标。她说"这些地方我都去过——但我不知道它们连在一起。"余烬站在桥中央，看着脚下流过的城市灯火，忽然觉得他这辈子做的最正确的决定不是不打大厂——是不将就。因为不将就，他才会站在这里。 |
| 情感基调 | 高潮——不是战斗的燃，是世界观展开的辽阔与感动 |
| 信息增量 | 桥连接所有城市节点；余烬的"数字游民"身份与卡牌世界天然契合——每个城市他都可以去，去了就有同类 |
| 预计字数 | ~1200 字 |

### 场景 3/3：桥的另一端（桥→现实过渡）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 桥的另一端→卡牌店，傍晚 |
| 出场人物 | 余烬、赵一行、高看、谢知白 |
| 场景目标 | 收束兔子洞三章；留下下一阶段的悬念 |
| 核心事件 | 从桥回来，四人聚在卡牌店。赵一行泡茶，高看转菩提，谢知白在笔记本上画桥的坐标图。赵一行说"你们三个——魂穿者、情报商、天梯宿敌——刚好凑一桌。"余烬问"一桌什么？"赵一行说"一桌能在这个世界活下来的人。"他顿了顿，"下一课——怎么在桥上不被风吹下去。"窗外梧桐叶落了一片，正好贴在玻璃柜上，像一张没翻面的卡牌。 |
| 情感基调 | 温暖、团队感初现、期待下一程 |
| 信息增量 | 赵一行暗示接下来的训练——在桥上行走需要精神力控制，是进阶课程 |
| 预计字数 | ~700 字 |

## 钩子设计

**开篇钩子**：高看把账本翻到最后一页。那一页没有字——只有一幅手绘的地图，线条歪歪扭扭，像小孩子画的藏宝图。但余烬认得其中一个标记：恒记卡牌店的位置。

**结尾钩子**：梧桐叶贴在玻璃柜上，叶脉像一张卡牌的背面纹路。余烬伸手拿起来，翻了个面——叶片背面有一个淡淡的漩涡印记。赵一行看了一眼，茶杯停在嘴边："有意思。这棵树，种了二十年，头一回有叶子翻出这个纹。"
