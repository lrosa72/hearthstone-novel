# 第4章 细纲 — "楼下的咖啡馆"

---

## 基本信息

| 项目 | 内容 |
|------|------|
| 章节标题 | 楼下的咖啡馆 |
| 预估字数 | 2000 ± 200 字 |
| 本章定位 | 节奏转换——从硬仗转入日常都市轻小说；带出李真意+谢知白；徐徐展开世界观 |

## 场景清单

### 场景 1/3：早晨的 routine（现实）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 合租房 → 楼下咖啡馆，早晨 |
| 出场人物 | 余烬、刘胖子（电话）、李真意（咖啡馆店员） |
| 场景目标 | 建立数字游民日常节奏；带出李真意这个"正常锚点" |
| 核心事件 | 余烬在家接外包，刘胖子来电扯淡（催他接活/吐槽）。下楼买咖啡，李真意记住他的口味，一句"今天怎么啦？你笑得比昨天勉强"点出她直觉极强 |
| 情感基调 | 松弛、温暖、毒舌包裹的舒适 |
| 信息增量 | 李真意=普通人但情绪读心（观众序列纯净版）；余烬在咖啡香里第一次觉得"不将就的日子"有具体温度 |
| 预计字数 | ~700 字 |

### 场景 2/3：能力的日常化（现实）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 楼下便利店/巷子，上午 |
| 出场人物 | 余烬、刘胖子（便利店）、苏新晖（微信/偶遇） |
| 场景目标 | 能力轻量化融入现实；轻描带出地下世界规则 |
| 核心事件 | 余烬用奥术飞弹帮刘胖子够下高货架/化解小麻烦，体会"润物无声"的改变。苏新晖微信提醒"清理者鼻子灵"——轻描带出势力存在 |
| 情感基调 | 轻松、惬意、"这日子能过" |
| 信息增量 | 能力在现实中可润物无声地使用；清理者是悬在头上的规则线 |
| 预计字数 | ~500 字 |

### 场景 3/3：炉石世界的老对手（炉石世界）

| 维度 | 内容 |
|------|------|
| 地点/时间 | 炉石酒馆，夜晚 |
| 出场人物 | 鱼刺（余烬）、谢知白·玉版白（天梯宿敌，法师控场） |
| 场景目标 | 带出谢知白（女主②）；徐徐展开炉石世界的社交层 |
| 核心事件 | 鱼刺在酒馆遇玉版白——游戏里死对头（鱼刺 vs 玉版白）真人初遇互怼。文白夹杂的毒舌才女 vs 脏牧毒舌，轻喜剧智斗不分胜负。玉版白点出"这酒馆里的人，各自有现实身份" |
| 情感基调 | 轻喜剧、旗鼓相当的怼、暗藏欣赏 |
| 信息增量 | 炉石世界"玩家"皆真人，有各自现实；天梯排名是圈子的社交货币 |
| 预计字数 | ~700 字 |

## 钩子设计

**开篇钩子**：余烬的早晨从一杯不该存在的拿铁开始——楼下新来的店员记得他的口味，还顺手读了他脸上的"将就"。

**结尾钩子**：玉版白端着酒杯看他："鱼刺，你现实里也是这副德行？"鱼刺一愣。她怎么知道"现实"这两个字？
