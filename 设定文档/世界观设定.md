# 世界观设定 — 《炉石之魂》

---

## 时代背景

| 维度 | 内容 |
|------|------|
| 时代类型 | 现代都市（2024年） |
| 文明程度 | 信息时代，互联网高度发达 |
| 社会结构 | 现代社会，存在隐性的"卡牌能力者"群体 |
| 时代特征 | 表面正常运转的都市，暗地里一小部分人拥有从"炉石世界"带回的能力 |

---

## 双世界架构

### 现实世界：现代都市

主角生活在普通的中国二线城市，和千千万万打工人一样为生存挣扎。

### 游戏世界：炉石传说 × 艾泽拉斯

主角魂穿到《炉石传说》的卡牌对战空间——一个将艾泽拉斯大陆浓缩为卡牌对决的异空间。这个世界有实体化的酒馆、竞技场、冒险模式副本。主角作为"玩家英雄"在其中战斗、成长。

### 双世界关系：能力具现

```
现实世界 ←→ 炉石世界

主角在两个世界间穿梭：
- 触发条件：现实中的十年炉石老玩家，达成"资格认证"后触发魂穿
  （余烬的触发时刻：牧师千胜达成，屏幕上的炉石logo发光）
- 魂穿方式：入睡后精神进入炉石世界，醒来回到现实
- 游戏经验转化：现实中十年的游戏知识（卡牌效果、combo、对局经验）全部保留，转化为真实战斗能力
- 在炉石世界获得的卡牌 → 在现实中可以具现使用
- 卡牌能力在现实中有限制（冷却时间、消耗精神力）
- 炉石世界的成长（等级、卡牌收集、竞技场战绩）→ 解锁更强的现实能力
```

## 力量体系：卡牌具现

### 卡牌等级

| 等级 | 名称 | 稀有度 | 现实具现限制 | 战力参考 |
|------|------|--------|-------------|----------|
| 1 | 基础卡 | 普通（白） | 24小时冷却 | 轻微超常（如短暂强化力量） |
| 2 | 稀有卡 | 稀有（蓝） | 12小时冷却 | 明显超常（如火球术、治疗） |
| 3 | 史诗卡 | 史诗（紫） | 6小时冷却 | 强力超常（如范围AOE、召唤随从） |
| 4 | 传说卡 | 传说（橙） | 1小时冷却 | 改变局势（如时间操控、传说随从具现） |
| 5 | 英雄卡 | 唯一（金） | 无冷却但消耗巨大 | 毁灭级（变身英雄、颠覆规则） |

### 职业体系

主角在炉石世界中可以选择职业，获得对应英雄技能：

| 职业 | 英雄技能 | 现实效果 |
|------|----------|----------|
| 法师 | 火焰冲击（2费打1） | 指尖点火、小型火球 |
| 战士 | 全副武装（2费+2护甲） | 皮肤硬化、伤害减免 |
| 盗贼 | 匕首精通（2费装备1/2武器） | 凭空召唤匕首、短距离瞬移 |
| 猎人 | 稳固射击（2费打英雄2） | 精准投掷、远程感知 |
| 术士 | 生命分流（2费抽牌掉2血） | 消耗生命力换取能力充能 |
| 牧师 | 次级治疗术（2费回2血） | 治疗小伤口 |
| 圣骑士 | 援军（2费召唤1/1报告兵） | 召唤小型随从 |
| 萨满 | 图腾召唤（随机图腾） | 布置小型结界/陷阱 |
| 德鲁伊 | 变形（2费+1攻+1甲） | 短暂身体强化 |

> 主角初始职业待定（建议法师/盗贼，更适合毒舌吐槽人设）

### 卡牌获取方式

1. **对战胜利**：在炉石世界赢得卡牌对战，获得对手卡牌的复制权
2. **冒险模式**：通关副本，获得特定卡牌
3. **竞技场**：连胜奖励，高风险高回报
4. **任务系统**：完成炉石世界的每日任务
5. **现实触发**：现实中完成特定挑战，解锁对应卡牌

## 现实世界设定

### 主角现状（开局）

- 姓名：陈墨（暂定，可改）
- 年龄：24岁
- 职业：待业/打零工（底层逆袭起点）
- 居住：城中村合租房
- 经济：欠房租两个月
- 关键道具：一款来路不明的《炉石传说》卡牌——这是他魂穿炉石世界的入口

### 卡牌能力者在现实中的处境

- 极少人知道卡牌能力的存在
- 存在一个隐秘的"卡牌能力者"圈子（地下竞技、黑市卡牌交易）
- 政府/财团对此有秘密研究
- 主角一开始完全不知道这些，随着能力增长逐渐卷入

## 核心规则

| 规则 | 说明 | 对剧情的影响 |
|------|------|-------------|
| 精神力限制 | 现实中具现卡牌消耗精神力，过度使用会昏迷甚至损伤大脑 | 防止主角无敌，制造紧张感 |
| 卡牌绑定 | 获得的卡牌绑定灵魂，不可交易但可被掠夺（输掉对战） | 制造冲突和赌注 |
| 冷却机制 | 卡牌在现实中具现后有冷却时间 | 战斗需要策略，不能无脑扔牌 |
| 隐藏原则 | 卡牌能力不能在普通人面前大规模暴露，否则引来"清理者" | 都市战斗必须低调 |
| 双世界时间差 | 炉石世界时间流速是现实的3倍（现实1小时=炉石3小时） | 主角可以在睡眠中获得大量游戏时间 |

## 势力分布

| 势力名称 | 类型 | 核心利益 | 对主角态度 |
|----------|------|----------|-----------|
| 普通社会 | 大众 | 不知道卡牌存在 | 无关 |
| 卡牌猎人 | 地下组织 | 狩猎其他能力者夺取卡牌 | 敌对 |
| 收藏家协会 | 隐秘组织 | 收集和研究传说卡牌 | 中立/可能合作 |
| 清理者 | 神秘势力 | 维持卡牌世界的隐秘性 | 敌对（如果主角暴露） |
| 天梯玩家 | 松散群体 | 在炉石世界中追求排名 | 中立/竞争 |
