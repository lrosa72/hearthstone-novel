# 炉石酒馆签到十年，不好意思我的卡牌真的能用

> 一个不将就的数字游民，因十年炉石老玩家的万胜资格认证魂穿炉石世界，获得卡牌具现能力。在都市暗面与炉石酒馆之间，他发现自己不是唯一的旅人——每座城市都有入口，每个入口都有同类。

## 写作铁律

1. **零容忍"不是A是B"**：叙事与心理独白一律禁用，对话保留
2. **白描直给**：名词动词干活，形容词滚蛋。王小波式列举+韩寒式直给
3. **无纯粹恶人**：冲突来自立场/信息差/命运，非邪恶
4. **双世界纹理一致**：酒馆=自走棋/昆特牌/麦酒/晶石灯；现实=麻将/象棋/爵士乐/快递
5. **卡牌对战方想式**：物理质感+完整动作链+空间感
6. **文艺但不假抒情**：宁可白描，不要华丽形容词
