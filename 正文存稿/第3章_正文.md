# 第3章 奥术飞弹

余烬盯着手心的"奥术飞弹"看了整整十分钟。

然后他做了一个很符合他性格的决定——先吃碗泡面。万一这是最后一次用人类的胃消化食物呢？

面吃到一半，他放下筷子，盯着卡牌。晨光里，法术图案的三颗飞弹还在慢慢转。

"来试试看。"他对自己说。

他按昨晚酒馆老板讲的来：凝神，想那张卡，把"要用它"的念头压进掌心。第一次，什么都没发生，指尖凉飕飕的，像对着没信号的遥控器使劲按。第二次，掌心冒了点蓝光，一闪就没，跟他手机电量一样不靠谱。第三次，他额头开始冒汗，像是有人从脑子里往外抽什么东西——

三颗蓝色的魔法飞弹，在掌心凝了出来。

它们悬浮着，比图样小一圈，但真真切切是实体，凉而轻，像攥着三颗会飞的冰。余烬还没来得及高兴，膝盖一软，整个人往下坠。不是累，是空——就像熬了三个通宵赶deadline，脑子被掏空了那种虚。

他扶着桌沿喘了半分钟。

"精神力，"他喃喃，"原来这玩意真有消耗量。"

手机震了。刘胖子："鱼刺！你昨儿说那游戏，我下了，叫啥来着？""炉石。"余烬回完，把飞弹散了。虚脱感还在，但他笑了——这东西，真的能用。

笑还没收住，楼道里传来脚步声，很重，带着股不耐烦。

门被敲响，不是苏新晖那种不紧不慢的节奏，是拳头砸门。

余烬开门。门外站着刀疤。

左脸那道疤在日光灯下更显眼，战斧没了虚影，换成手里真家伙——一把开过刃的实战斧。他上下打量余烬，像在确认没认错人。

"余烬。"刀疤说，"昨天炉石里赢我的，是你吧。"

余烬后背一凉。他记得清楚，昨天那场是在酒馆里打的，刀疤怎么找到现实的自己？

"卡牌猎人，"刀疤看出他的疑惑，"赢了的卡，带得回来；输了的场，也记得住人。你那张奥术飞弹，是我没看住的。今天我来——"

他没说完，斧头已经抡过来了。

余烬本能地往后一缩，斧风擦着鼻尖过去，削掉一撮刘海。他连滚带爬退回屋里，掌心一压，三颗飞弹又凝出来。

"就这三颗小玩意？"刀疤跨进门，战士的压迫感是真真切切的——他每进一步，余烬就退一步，后背已经抵上了墙。刀疤抬手，一张卡亮起，随从般的虚影在身后膨胀，是战士的狂暴随从，抡着比人还大的刀。

余烬知道正面刚不过。昨晚就被碾过一遍。

他没硬接。飞弹甩出去，没砸刀疤，而是炸在刀疤脚边那张狂暴随从的武器上——蓝光一炸，随从的刀偏了半寸。就这半寸，余烬从墙角滑出去，撞翻茶几拉开距离。

"跑？"刀疤嗤笑，一步跨过来，斧头横扫。他身后的狂暴随从也跟着动，铁刀劈空气的声儿尖得刺耳。

余烬第二颗飞弹炸在斧面，没挡住，但偏了方向。第三颗他留着，没扔——扔了也是送。刀疤的斧刃已经抵到他肩头，冰凉的钢铁贴着肩颈，他甚至能闻到斧头上铁锈混着汗的味道。

"昨天是运气。"刀疤说。

余烬盯着那把斧头。他手心里第三颗飞弹在抖——不是怕，是精神力快见底了，凝不稳。他忽然想起昨晚那场，他赢刀疤，靠的不是硬，是"卡那半拍"。

他没扔飞弹。他把最后一点精神力，全压进一个动作——假装手软，飞弹"啪"地散了，肩膀一垮，像认输。

刀疤的斧头顿了顿，本能地等他下一步。

就这顿的一瞬，余烬膝盖一顶刀疤胯下，借力往侧后方滚。刀疤没想到这小子认输是装的，斧头砍在地板，木屑飞溅。

余烬刚要爬起来，一道声音从门口砸下来。

"刀疤。"

不是喊，是压着火的沉沉一声。

刀疤整个人僵了。他握斧头的手没动，但肩膀塌了半分，像被无形的东西拽住。

门口站着个精瘦老头，皮手套，眼神锐利得像能剥人一层皮。余烬没见过他，但刀疤的反应说明了一切——这人，刀疤怕。

"住手。"老头说，"他是我看中的人。"

刀疤慢慢收回斧头。他没看老头，也没看余烬，只是腮帮子绷紧，半天才从牙缝里挤出来："师父。"

那声"师父"里没什么温情，倒像咽了口带刺的东西。余烬捕捉到刀疤垂下去的眼神——不是怕，是憋着一股没处发的气。

余烬的脑子"嗡"地一声。师父？

老头——后来他才知道叫老赵——没接刀疤这话，只对余烬说："第一次具现，就敢跟战士近身？胆子不小，脑子也还行。"他瞥了眼刀疤，"他欠我的，不在这儿算。你，"他指余烬，"想学怎么不送命，来街角卡牌店找我。"

说完，他转身走了。刀疤站在原地，盯着余烬看了几秒，那眼神复杂得很——有不服，有憋着的气，还有点说不清的东西。最后他也走了，没再动手。

余烬瘫在地上，肩颈的凉意半天散不掉。

一觉睡到傍晚。他是被饿醒的，也是被楼下的动静闹醒的——有人在楼下叫他名字。

他套上外套下楼，刚出单元门，一个人影靠在墙边。

苏新晖。马尾，利落，左手腕那道旧疤在路灯下泛白。她手里拎着便利店塑料袋，像只是下楼买个东西，顺路过来看看。

"你今天，"她抬眼，目光落在他手指上，"用过卡牌了？"

余烬的呼吸停了半拍。他今天用过奥术飞弹的事，除了刀疤和那个老赵，没第二个人知道——苏新晖怎么看出来的？

苏新晖没等他回答。她把塑料袋递过来，里面是两罐啤酒——跟他冰箱里那个牌子一样。

"下次用完，"她说，语气平淡得像在说"记得关火"，"别在楼道里留痕迹。清理者鼻子很灵。"她顿了顿，"还有，你刚才那套，太险了。战士近身，你那三颗飞弹不够看。"

她走了。余烬站在单元门口，手指还残留着奥术能量的灼热感，盯着她的背影，半天没动。

他的房东，靠在门框上，用一种审视同类的眼神，看了他一眼。

而他直到这一刻才彻底确认——这个看起来只是来收租的姑娘，和他，是一样的人。
